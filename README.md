# Villa Reservation Website (Laravel)

This project is a group-developed villa reservation web application created using the PHP Laravel framework.

## Features
- User registration and login system
- Villa listing and detailed pages
- Reservation/booking system
- Contact form
- Admin panel for managing villas and reservations

## My Contribution
I worked on backend development tasks, database connections and reservation logic. I implemented authentication processes and CRUD operations for some villa and booking features.

## Technologies
- PHP (Laravel)
- MySQL
- HTML
- CSS
- phpMyAdmin
